《Fireworks》

注意事項

・こちらを必ず読んでください。
　そして守れるお方のみがご利用になさってください。
・絶対にプラグイン(.jar)を他サイトに載せるのは禁止します。
・他サイトに掲載する場合はむつばのツイッターに連絡してください。
　 日本語の場合MCJFかMineAdd、日本語以外の場合SpigotMCに誘導するのをするのをおすすめします。

説明

・対応バージョン：1.16.x(1.16～1.16.5)
・動作確認バージョン：1.16.1
 ※1.16.1以外は正常動作するかは分かりません。
・対応言語：日本語、英語

サーバーにプラグインを入れる方法

Ⅰ，SpigotMCからダウンロードする！
Ⅱ，pluginフォルダーに入れる
Ⅲ，再起動・リロードをする！
Ⅳ，日本語にしたい人はconfig.ymlが生成されるのでその中のの方に日本語版があるので置き換える

コマンド

Command 説明
/fw help ヘルプの表示
/fw get random 数字 完全ランダムの生成された花火がもらえます
/fw get spawnrandom 数字 地面をクリックしたときにランダムの生成された花火が上に発射されます
/fw reload config.ymlをリロードする

コンソール

/fw help
実行不可✕

/fw get random 数字
実行不可✕

/fw get spawnrandom 数字
実行不可✕

/fw reload
実行可〇

権限

fireworks.* →defaultはOP
